<h3> Instruções e dicas </h3>

Pegue 30 moedas e mate todos os inimigos para vencer  
<h4> Bônus especiais:  </h4>
<p>10 moedas = Ganhe a habilidade de atacar o inimigo </p>    
<p>20 moedas = Aumente seu dano </p>   
<p>25 moedas = Paralise todos os inimigos e ganhe mais dano </p>

<h2> Como usar </h2>
<h3> Disponível para windows e android </h3>

   ```
   - Com o git instalado no computador, clone o repositório usando o:
   $ git clone https://github.com/e-medeiros1/suhiro_game.git 
   - Entre no diretório com:
   $ cd .\suhiro_game
   - Rode os seguintes comando antes: 
   $ flutter clean  
   $ flutter pub get  
   - Finalmente rode o jogo no dispositivo desejado:
   $ flutter run  
   ```
